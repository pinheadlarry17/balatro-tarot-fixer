# Balatro Tarot-fixer
Steammodded/Malverk mod for incorrectly numbered Tarot cards for Balatro

---

The Tarot-cards "Strength" and "Justice" are incorrectly numbered in-game. This mod fixes this through a texture-pack using Malverk.


## Requirements
- [Steammodded](https://github.com/Steamodded/smods)
- [Lovely](https://github.com/ethangreen-dev/lovely-injector)
- [Malverk](https://github.com/Eremel/Malverk)


## Installation

- Click on the green _"<> Code"_-button above
-  Select _"Download ZIP"_
-  Open the .zip file and drag and drop the folder named _"TarotFixer"_ into your Balatro mods folder (should be under _%appdata%/Balatro/Mods_, check the Steammodded GitHub-page for a more detailed explanation)
